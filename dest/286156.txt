 в Monitorul Oficial 
 
 Nr. 71-73 
    статья № : 183  Министерство финансов для руководства в практической работе, по импорту и поставкам товаров, услуг на территории республики для международных организаций в рамках соглашений, одной из сторон которых является Республика Молдова, согласно перечню международных организаций определенный в законе о бюджете на соответствующий год, с применением НДС по нулевой ставке согласно положений статьи 104 пункт d) Налогового Кодекса, Раздел III «Налог на добавленную стоимость» № 1415-ХШ от 17 декабря 1997, с последующими изменениями, доводит следующее. I. При приобретении товаров, услуг с территории республики пользователями проектов по технической помощи, местные поставщики поставляют товары, услуги пользователям проектов технической помощи, с применением НДС в размере 20% и выдают налоговую накладную. В случае строительных работ, осуществляемых для пользователей проектов по технической помощи, порядок применения НДС является следующим. Местные поставщики поставляют товары, услуги подрядчикам и субподрядчикам, которые осуществляют строительные работы для пользователей проектов технической помощи, с применением НДС и выдают налоговую накладную. Подрядчики и субподрядчики которые осуществляют строительные работы для пользователей проектов по технической помощи по материальным ценностям, услуги приобретенные для осуществления поставок услуг применяют НДС в размере 20% и выдают налоговую накладную. Пользователи проектов по технической помощи, по товарам, услугам приобретенные на территории республики от местных поставщиков с применением НДС по стандартной или сниженной ставке  для проектов технической помощи, могут требовать возмещение соответствующих сумм НДС согласно статьи 104 пункт d) Налогового кодекса, в следующем порядке: •	пользователи проектов по технической помощи, которые не зарегистрированы как плательщики НДС, в соответствии с Положением о порядке возмещения из бюджета сумм налога на добавленную сумму, уплаченных иностранными дипломатическими миссиями по товарам (работам, услугам), приобретенным для целей протокола, а также членами дипломатического корпуса для личного пользования, утвержденным Постановлением Правительства № 1032 от 26.12.2000 г. (Официальный монитор Республики Молдова, № 1-4 от 11.01.2001 г.). •	пользователи проектов по технической помощи, которые зарегистрированы  как плательщики НДС, в соответствии с подтверждающими документами, зачитывают суммы НДС уплаченные по приобретенным товарам, услугам. Если в течение налогового периода по НДС, у пользователя проекта по технической помощи, начисленные суммы НДС превышают  зачитанные суммы НДС, этот пользователь не может потребовать возмещения сумм НДС из бюджета. Если в течение налогового периода по НДС, у пользователя проекта по технической помощи, зачитанные суммы НДС превышают  начисленные суммы НДС, этот пользователь может потребовать возмещения сумм НДС из бюджета. В таком случае, Декларация по НДС заполняется следующим образом: -	в 19 строке указывается сумма НДС которая подлежит возмещению и составляет сумму НДС по соответствующим приобретениями, но не превышающая дебетовый остаток НДС; -	в 18 строке указывается разница межу дебетовым остатком НДС и суммы НДС указанной в 19 строке. Возмещение сумм НДС из бюджета осуществляется территориальными государственными налоговыми инспекциями только после налоговой проверки. Для этого плательщик НДС предъявляет государственной налоговой инспекции в радиусе которого обслуживается, заявление в свободной форме о возмещении сумм НДС из бюджета. Постановление о возмещение сумм НДС из бюджета издается территориальной государственной налоговой инспекцией. II. Представительства международных организаций в Республике Молдова, несмотря на то, что они определены в перечне, установленном Законом о бюджете на соответствующий год, квалифицированы и как миссии приравненных к дипломатическим миссиям в Республике Молдова, к которым относятся: 1.	Программа развития Организации Объединенных Наций (ПРООН). 2. Международный валютный фонд (МВФ). 3. Всемирный банк (ВБ). 4. Детский фонд Объединенных Наций (ЮНИСЕФ). 5. Управление Верховного комиссариата по делам беженцев ООН (УВКБООН). 6. Всемирная организация здравоохранения. 7. Представительство Международного Комитета Красного Креста. 8. Франкоязычное агентство. В случае, когда вышеназванные представительства международных организаций в Республике Молдова приобретают товары от местных поставщиков, услуги с территории республики с применением НДС по стандартным или уменьшенным ставкам, как в протокольных целях, так и для проектов технической помощи, которые они реализуют в Республике Молдова, эти представительства могут требовать возмещения соответствующих сумм НДС согласно ст. 104 пункт с) Налогового кодекса. Возмещение этих сумм НДС из бюджета осуществляется в соответствии с Положением о порядке возмещения из бюджета сумм НДС, уплаченных иностранными дипломатическими миссиями по товарам (работам, услугам), приобретенным для целей протокола, а также членами дипломатического корпуса для личного пользования, утвержденного Постановлением Правительства № 1032 от 26.12.2000 г. (Официальный монитор Республики Молдова, № 1-4 от 11.01.2001 г.). III. По импорту осуществленному пользователями проектов по технической помощи, реализованные международными организациями, определенными в законе о бюджете на соответствующий год, соответствено применяется НДС в размере 20%. Возмещение этих сумм НДС из бюджета осуществляется в соответствии с вышеуказанным порядком. Растаможивание импортов реализованных пользователями проектов технической помощи осуществляется Таможенным департаментом. IV. Доводим до сведения, что в соответствии с приложением к настоящему разъяснению, список проектов технической помощи, операторов и пользователей проектов, реализованных в Республике Молдова международными организациями, перечисленными в перечне, утвержденного законом о бюджете на соответствующий год, согласно которому поставки товаров, услуг для международных организаций осуществляются с применением НДС по нулевой ставке. В случае, когда произойдут изменения о завершении или начало новых проектов технической помощи, эти изменения дополнительно доводятся  до сведения территориальных государственных налоговых инспекций.
