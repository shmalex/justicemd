ПОЛОЖЕНИЕ   î Координационный совет по созданию   Национальной географической информационной системы   I. Общие положения   1. Координационный совет по созданию Национальной географической информационной системы (далее - совет) образован с целью координации действий по укреплению информационных ресурсов всех сфер жизнедеятельности общества, создания Государственной географической информационной системы, призванной решать задачи, связанные с обработкой данных по вопросам прогнозирования, планирования, управления, комплексного изучения окружающей среды, проектирования и реализации методов рационального использования, охраны и оценки природных ресурсов.   2. Совет состоит из представителей структур данной области Республики Молдова.   3. В своей деятельности совет руководствуется положениями Конституции Республики Молдова, законами Республики Молдова, указами Президента Республики Молдова, постановлениями и распоряжениями Правительства Республики Молдова.   Совет обобщает практику применения законодательства по вопросам обработки информации с использованием геоинформационных технологий для прогнозирования, планирования и управления природными ресурсами, вырабатывает предложения по совершенствованию этих технологий и вносит их на рассмотрение Правительства Республики Молдова.     II. Основные задачи совета   4. Основные задачи совета:   межведомственная координация действий по созданию Государственной географической информационной системы;   координация деятельности отраслевых структур по вопросам интеграции данных по созданию Государственной географической информационной системы. Разрешение возможных разногласий при создании Государственной географической информационной системы;   координация работ по созданию пространственной базы данных;   определение источников финансирования для создания Государственной географической информационной системы;   установление сроков создания Государственной географической информационной системы;   контроль качества данных.     III. Компетенция cовета   5.  В компетенцию cовета входит:   разработка проектов законодательных и нормативных актов по стандартизации форм базы данных, классификаторов картографических объектов, цифровой модели местности, базовых программ и способов информационного обмена;   рассмотрение технических проектов разработки банков данных цифровых карт;   утверждение системы стандартов баз данных, технических параметров тематических слоев, пригодных для разработки основной цифровой карты Республики Молдова.   рассмотрение и согласование проектов концепций отраслевых географических информационных систем.          [Приложение 2, пкт.5, абз.5 введен ПП676 от 11.07.05, МО95-97/15.07.05 ст.733]       IV. Организация деятельности cовета   6. Председатель cовета:   организует деятельность cовета;   устанавливает место и дату проведения заседаний;   распределяет функции между заместителями председателя, руководит их деятельностью.   7. Работа секретаря:   секретарь cовета организует его заседания, выполняет другие функции по распоряжению председателя cовета и его заместителей;   ведет реестр учета материалов, полученных cоветом;   ведет реестр учета материалов, утвержденных cоветом.   8. Заседание cовета считается правомочным, если на нем присутствует не менее половины членов cовета.   9. Заседания cовета созываются по необходимости.    10.  Решения cовета оформляются протоколами.
