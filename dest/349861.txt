Описание предметов форменной одежды  сотрудников Национального центра по борьбе с коррупцией  I. Форменная одежда для высшего начальствующего состава  1. Фуражка из шерстяной ткани светло-серого цвета Фуражка состоит из донышка овальной формы, тульи, состоящей из четырех частей, околыша красного цвета, козырька и шнура золотистого цвета. Спереди на околыше фуражки по центру находятся кокарда и шитье золотистого цвета в виде дубовых листьев. По краю донышка и верхнему краю околыша – канты красного цвета. Козырек твердый, обтянут сверху лакированной черной кожей, вдоль наружного края козырька симметрично в два ряда расположена отделка в виде четырех дубовых ветвей, каждая с  четырьмя листьями, из золотистого металла, шириной 20 мм. Для сохранения формы донышка в фуражку вставлен металлический каркас. Фуражка на шелковой подкладке.  2. Фуражка из шерстяной ткани темно-синего цвета Описание соответствует описанию фуражки из шерстяной ткани светло-серого цвета (пункт 1).  3. Шапка из натурального каракуля серого цвета Шапка состоит из донышка овальной формы, цилиндрического окола, козырька, наушников и ремешка. Все детали выполнены из натурального каракуля серого цвета. На передней стороне окола по центру крепится кокарда. Шапка на стеганой хлопчатобумажной подкладке на ватине.  4. Китель из шерстяной ткани светло-серого цвета Китель состоит из полочек, спинки, рукавов и воротника. Полочки состоят из двух симметричных частей. Застежка двубортная в три уровня, на две петли и пуговицы. Острые углы лацканов направлены к швам рукавов. Ниже линии талии, параллельно ей, расположены два прорезных кармана с клапанами. Спинка состоит из двух симметричных частей со швом посередине. В швах стачивания полочки и спинки – две шлицы. Рукав втачной, двухшовный, с обшлагом. Воротник отложной с прямыми углами. На углах воротника и обшлагах шитье золотистого цвета в виде виноградной лозы. По линии притачивания обшлага и краю воротника втачан кант красного цвета. Китель на хлопчатобумажной подкладке.  5. Китель из шерстяной ткани темно-синего цвета Китель состоит из полочек, спинки, рукавов и воротника. Полочки состоят из двух симметричных частей. Застежка однобортная на четыре петли и пуговицы. Острые углы лацканов направлены к швам рукавов. Ниже линии талии, параллельно ей, расположены два прорезных кармана с клапанами. Спинка состоит из двух симметричных частей со швом посередине, заканчивающимся шлицей. Рукав втачной, двухшовный, с обшлагом. Воротник отложной с прямыми углами. На углах воротника шитье золотистого цвета в виде виноградной лозы. По линии притачивания обшлага и краю воротника втачан кант красного цвета. Китель на хлопчатобумажной подкладке. 6. Пальто из натуральной кожи черного цвета Пальто состоит из полочек, спинки, рукавов, воротника и пояса с пряжкой. Полочки состоят из двух симметричных частей. Застежка двубортная на четыре петли и пуговицы. Лацканы с прямыми углами. Ниже линии талии под углом расположены два прорезных кармана, заделанных листочками. Спинка состоит из верхней и нижней частей со швом по линии талии. Верхняя часть спинки состоит из декоративной кокетки, заднего бокового клина и средней части спинки. Нижняя часть спинки состоит из двух частей со швом посередине, заканчивающимся шлицей. Шлица застегивается на две кнопки  с внутренней стороны. В области талии размещены три шлевки для пояса: две в боковых швах и одна – в шве спинки. Рукав втачной, двухшовный. Низ рукава отделан декоративным хлястиком, свободный конец которого закреплен пуговицей. Воротник отложной с отрезной стойкой, с прямыми углами. На него надевается съемный меховой воротник из натурального каракуля. Съемный воротник пристегивается к стойке основного воротника на пять петель и пуговиц и липкой лентой по отлету воротника в пяти местах. Пальто имеет отлетную кокетку, вшитую в плечевые и боковые швы и швы притачивания рукавов. Свободные углы кокетки закреплены пуговицами. Пальто на шелковой подкладке. Дополнительно предусмотрена съемная утепляющая безрукавная подкладка из натурального меха, пристегивающаяся с помощью застежки-молнии по внутреннему краю подбортов.  7. Куртка утепленная из ткани темно-синего цвета Куртка состоит из полочек, спинки, рукавов, воротника и пояса с пряжкой. Полочки состоят из двух симметричных частей, застегивающихся на застежку-молнию, поверх которой расположена накладная планка с застежкой на пять кнопок. Ниже линии талии под углом расположены два прорезных кармана, заделанных листочками. Спинка состоит из верхней части (кокетки) и нижней части. В области талии размещены три шлевки для пояса: две в боковых швах и одна – в шве спинки. Рукав втачной, двухшовный. Низ рукава с внутренней стороны обработан эластичным манжетом из двойного трикотажа. Погон с одного края вшивается в шов рукава, а свободным концом пристегивается кнопкой к плечевому шву. Воротник отложной с отрезной стойкой. На него надевается съемный меховой воротник из натурального каракуля. Съемный воротник пристегивается к стойке основного воротника на пять петель и пуговиц и липкой лентой по отлету воротника в пяти местах. В шве притачивания воротника к горловине с внутренней стороны находится карман для капюшона. Куртка на шелковой подкладке. Дополнительно предусмотрена съемная утепляющая безрукавная подкладка из натурального меха, пристегивающаяся с помощью застежки-молнии по внутреннему краю подбортов. На подкладке находятся два внутренних кармана, застегивающихся при помощи липкой ленты.  8. Плащ из ткани темно-синего цвета Плащ состоит из полочек, спинки, рукавов, воротника и пояса с пряжкой. Полочки состоят из двух симметричных частей, застегивающихся на четыре петли и пуговицы. Острые углы лацканов направлены к швам рукавов. Ниже линии талии под углом расположены два прорезных кармана, заделанных листочками. Спинка состоит из двух симметричных частей со швом посередине, заканчивающимся шлицей. В области талии размещены три шлевки для пояса: две в боковых швах и одна – в шве спинки. Рукав втачной, двухшовный. Низ рукава отделан декоративным хлястиком, свободный конец которого закреплен пуговицей. Воротник отложной с прямыми углами. Плащ имеет отлетную кокетку, вшитую в плечевые и боковые швы и швы притачивания рукавов. Нижний срез кокетки окантован, свободные углы кокетки закреплены пуговицами. Плащ на шелковой подкладке. Дополнительно предусмотрена съемная утепляющая безрукавная подкладка из сукна, пристегивающаяся с помощью  петель и пуговиц по внутреннему краю подбортов.  9. Свитер трикотажный полушерстяной темно-синего цвета Свитер состоит из полочки, спинки и рукавов. Полочка и спинка представляют собой единую деталь из гладкого трикотажа. Полочка с V-образным вырезом, край которого обработан вязаной кетлевочной бейкой шириной 2,5–3,0 см. В области плеч и локтей накладные кокетка и налокотники из плащевой ткани. Рукав втачной, одношовный. Погоны с одного края вшиваются в швы притачивания рукавов, а свободными концами пристегиваются пуговицей к плечевому шву. Погоны выполнены из плащевой ткани. Низ свитера и рукавов обработан эластичными вязаными поясом и манжетами из двойного трикотажа.  10. Рубашка из искусственной шелковой ткани белого цвета Рубашка состоит из полочек, спинки, рукавов и воротника. Полочки состоят из двух симметричных деталей, застегивающихся на шесть петель и пуговиц. В верхней части полочек расположены два накладных кармана с клапанами, застегивающимися на петли и пуговицы. Спинка состоит из верхней части (кокетки) и нижней части. В шов кокетки заложены две симметрично расположенные складки, направленные к швам притачивания рукавов. Рукав втачной, двухшовный, с манжетом, застегивающимся на одну петлю и пуговицу. Воротник отложной с отрезной стойкой. Рубашка с притачным поясом, застегивающимся по центру на две петли и пуговицы. В области боковых швов в пояс вдета эластичная лента, закрепленная  тремя отделочными строчками.  11. Рубашка из искусственной шелковой ткани серо-голубого цвета Описание соответствует описанию рубашки из искусственной шелковой ткани белого цвета (пункт 10).  12. Рубашка из искусственной шелковой ткани серо-голубого цвета с коротким рукавом и рубашка из искусственной шелковой ткани белого цвета с коротким рукавом Описание соответствует описанию рубашки из искусственной шелковой ткани белого цвета (пункт 10). Особенности: рукав короткий до уровня локтя. Низ рукава заканчивается цельнокроеным манжетом.  13. Галстук из искусственной шелковой ткани темно-синего цвета и галстук из искусственной шелковой ткани светло-серого цвета Галстук состоит из основной части, узла и шейки. Основная часть в форме вытянутой трапеции заканчивается острым углом. Узел постоянный треугольной формы. Шейка из двух половинок прямоугольной формы застегивается на воротнике на крючок и петлю.  14. Брюки из шерстяной ткани светло-серого цвета Брюки состоят из передних и задних половинок и пояса. Передние половинки состоят из двух симметричных частей. В среднем шве расположена застежка на четыре-пять петель и пуговиц. В боковых швах находятся два кармана. Прилегание по линии талии достигается с помощью складок, размещенных по линии стрелки и направленных к боковым швам. Задние половинки состоят из двух симметричных частей. Вверху на правой половинке расположен карман с клапаном, застегивающимся на петлю и пуговицу. Прилегание по линии талии достигается с помощью двух вытачек, симметрично расположенных на обеих частях. В боковые швы втачан кант красного цвета. На расстоянии 10 мм от канта по обе стороны расположены лампасы из сукна красного цвета. Ширина лампаса 25 мм. Пояс втачан в шов по линии талии вместе с семью шлевками для ремня. Пояс застегивается на петлю, пуговицу и крючок. Низки брюк с внутренней стороны обработаны специальной лентой.  15. Брюки из шерстяной ткани темно-синего цвета Описание соответствует описанию брюк из шерстяной ткани светло-серого цвета (пункт 14).  16. Кашне трикотажное полушерстяное белого цвета Кашне размером 130×25 см с бахромой на концах.  17. Кашне трикотажное полушерстяное темно-синего цвета Описание соответствует описанию кашне трикотажного полушерстяного белого цвета (пункт 16). 18. Перчатки из хлопчатобумажной трикотажной ткани белого цвета Перчатки состоят из корпуса с четырьмя пальцами, большого пальца и межпальцевых полосок.  19. Перчатки из натуральной кожи черного цвета Перчатки состоят из корпуса с четырьмя пальцами, большого пальца и межпальцевых полосок. В боковых швах перчаток имеются разрезы. Нижний край и разрез перчаток окантованы кожей. Перчатки на подкладке из трикотажного полотна.  20. Туфли из натуральной кожи серого цвета Туфли состоят из узлов верха и низа. Узел верха состоит из наружных деталей и подкладки. Наружными деталями верха являются союзка и берцы. Высота берца – 6,5 см. Детали подкладки изготовлены из натуральной кожи.     Узел низа состоит из наружных, внутренних и промежуточных деталей. Наружными деталями низа являются подошва и каблук, внутренними – стельки и полустельки, промежуточными – супинатор, задник и подносок. Узел низа крепится к узлу верха клеевым методом. Туфли крепятся на стопе при помощи эластичной ленты, расположенной между берцами под союзкой.  21. Туфли из натуральной кожи черного цвета Описание соответствует описанию туфель из натуральной кожи серого цвета (пункт 20).  22. Ботинки с высокими берцами из натуральной кожи черного цвета Ботинки состоят из узлов верха и низа. Узел верха состоит из наружных деталей и подкладки. Наружными деталями верха являются перед, берцы, задний наружный ремень, штафирка и язычок. Высота берца – 21 см. Детали подкладки изготовлены из натуральной кожи. Узел низа состоит из наружных, внутренних и промежуточных деталей. Наружными деталями низа являются литая подошва с каблуком, внутренними – стельки, полустельки и вкладные стельки, промежуточными – супинатор, жесткие задник и подносок. Узел низа крепится к узлу верха клеевым (клеепрошивным) методом. Ботинки крепятся на стопе при помощи блочек и шнурков. 23. Полусапоги утепленные из натуральной кожи черного цвета Полусапоги состоят из узлов верха и низа. Узел верха состоит из наружных деталей и подкладки. Наружными деталями верха являются перед и берцы. Высота берца – 21 см. Детали подкладки изготовлены из натурального меха. Узел низа состоит из наружных, внутренних и промежуточных деталей. Наружными деталями низа являются литая подошва с каблуком, внутренними – стельки, полустельки и вкладные стельки, промежуточными – супинатор, задник и подносок. Узел низа крепится к узлу верха клеевым методом. Полусапоги застегиваются на застежку-молнию, расположенную с внутренней стороны. II. Форменная одежда для старшего и среднего начальствующего и младшего состава  24. Фуражка из шерстяной ткани темно-синего цвета Описание соответствует описанию фуражки из шерстяной ткани светло-серого цвета (пункт 1). Особенности: на передней части околыша отсутствует шитье золотистого цвета. Для полковников вдоль наружного края козырька фуражки симметрично в один ряд расположена отделка в виде двух дубовых ветвей, каждая с четырьмя листьями, из золотистого металла шириной 20 мм. Козырек фуражки для остального начальствующего и для младшего состава – без отделки. Цвет околыша и канта – темно-синий. 25. Описание знаков различия сотрудников Национального центра по борьбе с коррупцией Устанавливаются следующие знаки различия: 1) кокарда; 2) эмблема для петлиц и погон; 3) нарукавный шеврон; 4) погоны с галунами и звездами; 5) пуговицы; 6) шитье на фуражке; 7) козырек фуражки; 8) зажим для галстука.  1. Кокарда Кокарда представляет собой овальный щит, пересеченный червленью и лазурью, в центре которого рельефное золотое изображение Государственного герба Республики Молдова. Щит расположен на золотой 40-лучевой овальной звезде. Высота кокарды  40 мм, ширина – 30 мм. Кокарда изготавливается из цветного металла и покрывается эмалью. С оборотной стороны припаяна пара металлических усиков, при помощи которых кокарда крепится в передней части фуражки, шапки, пилотки или берета. Центр кокарды должен находиться над переносицей.  2. С оборотной стороны припаяна пара металлических усиков, при помощи которых эмблема крепится на воротнике или погоне.  3. Нарукавный шеврон изготавливается шитьем или печатью по полотну. Высота нарукавного шеврона 90 мм, ширина – 70 мм. 4. Погоны для старшего и среднего начальствующего состава, младшего состава изготовлены из ткани, из которой выполнена соответствующая форменная одежда, вшивные, шириной 50 мм. Вдоль погона посередине нашит галун, золотой, шириной 20 мм для начальствующего состава и 13 мм для младшего состава. Диаметр звезды для высшего начальствующего состава – 28 мм, для старшего начальствующего состава – 20 мм, для среднего начальствующего состава и младшего состава – 13 мм.
