Министерству финансов выделить из общегосударственных доходов государственного бюджета районному совету Окница один миллион леев для реконструкции Дома-музея К. Стамати в селе Окница с включением соответствующих средств при внесении поправок в государственный бюджет на 2005 год в приложение № 4.   2. Районному совету Окница в качестве получателя выделенных средств разработать в установленном порядке документы, необходимые для финансирования указанных расходов.   3. Министерству финансов профинансировать работы по реконструкции Дома-музея К. Стамати в селе Окница по мере представления документов, подтверждающих понесенные расходы.
