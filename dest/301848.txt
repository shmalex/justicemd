    HGO419/2002 Внутренний номер:  301848  Varianta în limba de stat   Карточка документа   Республика Молдова   ПРАВИТЕЛЬСТВО   ПОСТАНОВЛЕНИЕ Nr. 419 от  05.04.2002   о выделении финансовых средств, необходимых для осуществления Республикой Молдова сопредседательства в рамках 3-го Рабочего Стола Пакта стабильности для Юго-Восточной Европы  Опубликован : 11.04.2002 
 в Monitorul Oficial 
 
 Nr. 50-52 
    статья № : 506    Исходя из принятия Республикой Молдова, начиная с 1 июля 2002 г., сопредседательства в рамках 3-го Рабочего Стола Пакта стабильности для Юго-Восточной Европы, Правительство Республики Молдова ПОСТАНОВЛЯЕТ:  Министерству финансов выделить Министерству иностранных дел из резервного фонда Правительства:  230000 леев согласно представленной смете расходов в целях организации пленарного заседания 3-го Рабочего Стола Пакта стабильности для Юго-Восточной Европы, а также проведения двух координационных совещаний председателей рабочих групп в рамках 3-го Рабочего Стола Пакта стабильности для Юго-Восточной Европы;  необходимые суммы для эффективного участия и осуществления сопредседательства в рамках 3-го Рабочего Стола Пакта стабильности в различных заседаниях, конференциях, симпозиумах и т. д., проводимых в странах - участницах по проблемам, относящимся к функциональной компетенции 3-го Рабочего Стола Пакта стабильности для Юго-Восточной Европы, по представлении необходимой сметы расходов для каждого конкретного случая, утвержденной постановлением Правительства;  100000 леев согласно смете расходов для снабжения электронными средствами связи и множительной техникой в целях обеспечения некоторых координационных мероприятий и постоянного обмена информацией между сопредседателем и государствами - участниками Пакта стабильности. Средства, выделенные для этого, будут возмещены при последующем уточнении государственного бюджета.
