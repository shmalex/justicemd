    статья № : 1112  В соответствии с Постановлением Правительства №  632 от 14 июля 2010 г. «О создании Национальной комиссии по ликвидации последствий наводнения» (Официальный монитор Республики Молдова, 2010 г., № 124-125, ст.714), с последующими изменениями и дополнениями,   Национальная комиссия по ликвидации последствий наводнения  приняла решение: 1. Министерству финансов  выделить со специального счета для  оказания помощи пострадавшим финансовые средства, из которых: Агентству земельных отношений и кадастра  в сумме 2500 тыс. леев для покрытия расходов, связанных с выполнением кадастровых работ по строительству жилых домов для пострадавших из района Хынчешть; районному  совету Хынчешть: 300 тыс. леев – для покрытия расходов, связанных с обеспечением питания пострадавших из района Хынчешть; 912,7 тыс. леев – в целях возмещения стоимости  сельскохозяйственной продукции, выращенной на участках, использованных под строительные площадки; 1826,0 тыс. леев –  для возмещения сельскохозяйственных потерь; районному совету Унгень: 85,43 тыс. леев – для пострадавших семей, дома которых будут отремонтированы; 543,19 тыс. леев – для  пострадавших семей, дома которых были разрушены и которые не воспользовались приобретением или постройкой новых домов; 494,7 тыс. леев – для возмещения сельскохозяйственного ущерба; районному  совету Кантемир: 219,6 тыс. леев – для пострадавших семей,  дома которых были разрушены и которые не воспользовались приобретением или постройкой новых домов; 1843,0 тыс. леев – для возмещения сельскохозяйственного ущерба; районному совету Анений Ной – 50,45 тыс. леев – для возмещения сельскохозяйственного ущерба; районному совету Бричень – 175,76 тыс. леев –  для возмещения сельскохозяйственного ущерба; районному совету Кахул – 345,59 тыс. леев – для возмещения сельскохозяйственного ущерба; районному совету Единец – 67,51 тыс. леев – для возмещения сельскохозяйственного ущерба; районному совету Леова – 3 тыс. леев – для возмещения сельскохозяйственного ущерба; районному совету Рышкань – 116,64 тыс. леев – для возмещения сельскохозяйственного ущерба; районному совету Шолдэнешть  - 279,75 тыс. леев – для возмещения сельскохозяйственного ущерба; районному совету  Штефан  Водэ – 86,6 тыс. леев – для возмещения сельскохозяйственного ущерба. 2. Настоящее решение  опубликовать в Официальном мониторе Республики Молдова.  Премьер-министр, председатель Комиссии                                	Владимир ФИЛАТ Зам.
