    статья № : 319  Правительство ПОСТАНОВЛЯЕТ: 1. Принять к сведению проект Кредитного соглашения между Правительством Республики Молдова и Международной ассоциацией развития по проекту «е-Превращение правления». 2. Инициировать переговоры по проекту Кредитного соглашения между Правительством Республики Молдова и Международной ассоциацией развития по проекту «е-Превращение правления». 3. Создать группу по ведению переговоров с Международной ассоциацией развития по проекту Кредитного соглашения между Правительством Республики Молдова и Международной ассоциацией развития по проекту «е-Превращение правления» в следующем составе:  НЕГРУЦА  Вячеслав    -    министр финансов, председатель группы по ведению переговоров МОКАН Стела               -    исполнительный директор, Центр электронного управления ШКОЛА Дона               -    заместитель министра информационных технологий и связи ЧУРЯ Лукреция            -    начальник отдела внешней помощи управления политик, стратегического планирования и внешней помощи, Государственная канцелярия МАТВЕЕВА Елена        -    начальник Главного управления государственного долга, Министерство финансов 4. Группе по ведению переговоров провести переговоры с Международной ассоциацией развития в целях выделения финансирования в кратчайшие сроки и на взаимовыгодных условиях. 5. Предоставить полномочия господину Вячеславу НЕГРУЦЕ, министру финансов, для ведения переговоров и парафирования текста проекта Кредитного соглашения между Правительством Республики Молдова  и Международной ассоциацией развития по проекту «е-Превращение правления».
