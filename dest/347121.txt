 в Monitorul Oficial 
 
 Nr. 56-59 
    статья № : 241  Правительство ПОСТАНОВЛЯЕТ: 1. Принять к сведению проекты соглашений, заключенных путем обмена нотами, между Правительством Японии и Правительством Республики Молдова для финансирования проекта «Улучшение медицинских услуг». 2. Инициировать переговоры по проектам соглашений, заключенных путем обмена нотами, между Правительством Республики Молдова и Правительством Японии для финансирования проекта «Улучшение медицинских услуг». 3. Образовать группу по ведению переговоров по проектам соглашений, заключенных путем обмена нотами, между Правительством Японии и Правительством Республики Молдова для финансирования проекта «Улучшение медицинских услуг» в следующем составе:  НЕГРУЦА Вячеслав           - министр финансов, руководитель группы по ведению переговоров УСАТЫЙ Андрей              - министр здравоохранения КОДРЯНУ Руслан             - начальник главного управления координирования политик, внешней помощи и реформы центрального публичного управления, Государственная канцелярия МАТЕЙ Андрей                 - начальник управления бюджета, финансов и страхования, Министерство здравоохранения МАТВЕЕВА Елена             - начальник главного управления  государственного долга, Министерство финансов ПЕРЕБИНОС Каролина    - начальник управления международных договоров, главное управление международного права, Министерство иностранных дел и европейской интеграции. 4. Группе по ведению переговоров осуществлять переговоры с представителями Правительства Японии для выделения финансирования в кратчайшие сроки и на взаимовыгодных условиях. 5. Уполномочить господина Вячеслава НегруцУ, министра финансов, руководителя группы по ведению переговоров, для подписания протокола переговоров и других документов, касающихся данных переговоров.
