    Промульгирован : 28.03.1997    АНУЛИРОВАН 29.09.1999  В целях выполнения   условий  Всемирного  банка  по   предоставлению  кредита  в  целях микрокредитования частного сектора села  Пpавительство  Pеспублики Молдова ПОСТАНОВЛЯЕТ:  1. Создать при  Министерстве финансов Агентство по внедрению проекта  финансирования  села, основные функции которого заключаются в разработке  законодательной  базы деятельности ассоциаций кредитования и сбережений,  а  также  в осуществлении мониторинга, консультаций и  обучения  кадров,  необходимых  для нормального функционирования системы  микрокредитования  частного сектора села.  Срок деятельности   указанного  агентства  продлится  до   внедрения  проекта финансирования села.  2. Утвердить   решение   совета  по  внедрению  данного  проекта   о  назначении   г-на   Виктора  Кирияка  ответственным   координатором   и,  соответственно, директором Агентства по внедрению проекта финансирования  села.  3. Министерству  финансов  в 10-дневный срок утвердить положение  об  Агентстве  по внедрению проекта финансирования села и другие  документы,  необходимые для его функционирования.  4. В целях выполнения работ по внедрению проекта финансирования села  деятельность  Агентства  по внедрению проекта финансирования села  будет  финансироваться   из   аванса,  предоставленного  Всемирным   банком   в  соответствии  с  письмом-соглашением N РЗ140-MD от 4 февраля 1997 г.,  а  также из других пожертвований для этого проекта.
