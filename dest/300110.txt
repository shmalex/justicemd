    статья № : 245    Правительство ПОСТАНОВЛЯЕТ:   1. Министерству финансов выделить из резервного фонда Правительства районному совету Сорока 200 тыс. леев для капитального ремонта школы села Окланда, район Сорока, и районному совету Штефан Водэ 100 тыс. леев для завершения строительства котельной для лицея им. Б.П.Хашдеу из села Олэнешть, район Штефан Водэ.   2. Районному совету Сорока и районному совету Штефан Водэ в качестве заказчиков выделенных средств разработать в установленном порядке документы, необходимые для финансирования указанных расходов.   3. Министерству финансов профинансировать работы по капитальному ремонту школы села Окланда, район Сорока, и по завершению строительства котельной лицея им. Б.П.Хашдеу из села Олэнешть, район Штефан Водэ, по мере представления документов, подтверждающих осуществление расходов.
